## Welcome to Flowise

Flowise is an open-source, low-code platform that empowers you to build AI agents and LLM workflows visually. With its intuitive drag-and-drop interface, you can create sophisticated AI applications without deep technical expertise.

## Architecture

Flowise is built as a full-stack application with three main modules:

- **Server**: Node.js backend serving API logic with Express
- **UI**: React frontend providing the visual workflow builder
- **Components**: Third-party integrations and custom nodes for LLMs, vector stores, tools, and more
