## GitHub Discussions

Our GitHub Discussions is organized into several categories:

### Q&A

Ask questions and get help from the community and maintainers. Before posting:

1. Search existing discussions to see if your question has been answered
2. Provide clear details about your issue or question
3. Include relevant code snippets, error messages, or screenshots

[Browse Q&A →](https://github.com/FlowiseAI/Flowise/discussions/categories/q-a)

### Show and Tell

Share your chatflows and AI applications built with Flowise:

- Export your chatflow as JSON
- Include screenshots of your flow
- Describe the use case and any interesting approaches
- Share tips and lessons learned

[View Showcases →](https://github.com/FlowiseAI/Flowise/discussions/categories/show-and-tell)

### Ideas

Propose new features and improvements:

- New component integrations
- UI/UX enhancements
- Performance optimizations
- Developer experience improvements

[Share Ideas →](https://github.com/FlowiseAI/Flowise/discussions/categories/ideas)

## Contributing to the Community

### Help Others

One of the best ways to contribute is by helping other community members:

- Answer questions in Discord and GitHub Discussions
- Share your knowledge and experience
- Review and test pull requests
- Provide feedback on new features

### Share Your Work

<Steps>
  <Step title="Build Something Cool">
    Create an interesting chatflow or integration using Flowise
  </Step>
  
  <Step title="Document It">
    Take screenshots and export your chatflow as JSON
  </Step>
  
  <Step title="Share">
    Post in the Show and Tell section with:
    - Description of what you built
    - Use case and benefits
    - Challenges and solutions
    - Screenshots and JSON export
  </Step>
</Steps>

### Report Issues

Help improve Flowise by reporting bugs:

1. Check if the issue already exists
2. Provide detailed reproduction steps
3. Include environment information (OS, Node version, etc.)
4. Add error messages and logs

[Report an Issue →](https://github.com/FlowiseAI/Flowise/issues/new/choose)

## Community Guidelines

<Warning>
All community interactions are governed by our [Code of Conduct](/resources/contributing#code-of-conduct). Please be respectful and constructive.
</Warning>

### Best Practices

- **Be respectful**: Treat others with kindness and respect
- **Be helpful**: Share your knowledge and experience
- **Be patient**: Remember that everyone has different skill levels
- **Be constructive**: Provide actionable feedback
- **Be professional**: Keep discussions on-topic and appropriate
